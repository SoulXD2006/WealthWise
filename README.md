# WealthWise

WealthWise is a modern fintech platform designed to help users manage their personal finances, track investments, monitor live market data, and receive AI-powered financial insights through a unified dashboard.

## Features

* Interactive financial dashboard with portfolio analytics
* Investment portfolio tracking and asset allocation visualization
* Market overview with stock performance monitoring
* Financial news and market updates
* AI-powered financial assistant for portfolio and spending insights
* Budget tracking and expense management
* Personalized financial alerts and recommendations
* Customizable user settings and preferences
* Secure authentication and onboarding experience

## Tech Stack

### Frontend

* React
* TypeScript
* Tailwind CSS
* Vite
* Recharts

### Planned Backend

* Spring Boot
* PostgreSQL
* JWT Authentication
* Google OAuth

### Integrations

* Gemini API (AI Financial Assistant)
* Finnhub API (Market Data)
* News API (Financial News)

## Vision

WealthWise aims to serve as a personal financial intelligence platform that combines budgeting, investing, market monitoring, and AI-driven insights in a single seamless experience.

## Status

Currently under active development.

The frontend experience has been completed, including portfolio management, market analytics, budgeting tools, financial news, AI assistant interfaces, onboarding flows, and application settings. Backend services, authentication, database integration, live market data, and AI-powered analytics are currently being implemented.
